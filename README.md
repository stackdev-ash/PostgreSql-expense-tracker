# PostgreSQL Expense Tracker API

A simple Expense Tracker REST API built with Node.js, Express, TypeScript, and PostgreSQL.

## Features

* CRUD Operations
* Search & Filter
* Sorting
* Pagination
* Expense Statistics
* Category Analytics

## Tech Stack

* Node.js
* Express.js
* TypeScript
* PostgreSQL

## Run Locally

```bash
npm install
npm run dev
```

## Learning

Built to learn PostgreSQL integration with Express, including SQL queries, filtering, pagination, aggregate functions, and analytics.
